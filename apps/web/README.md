# AgentVault Web

This is the frontend app for AgentVault.

## Setup

Install workspace dependencies from the repository root:

```bash
pnpm install
```

## Start the app

From the repository root:

```bash
pnpm --filter @agentvault/web dev
```

Or from this folder:

```bash
pnpm install
pnpm dev
```

## Build

From the repository root:

```bash
pnpm --filter @agentvault/web build
```

## Typecheck

From the repository root:

```bash
pnpm --filter @agentvault/web typecheck
```
